Exploratory testing

Testing a WIP design

  • Exploratory testing is particularly useful when a project is still in the design phase. With decisions still in flux, an adaptive testing approach helps guide the creation process.
  • While a design is still in progress, you may not be able to test it with actual assistive technology (AT). Tools like Figma are slowly getting better at making their prototypes more accessible. Figma prototypes can support basic keyboard navigation, for example. However, it doesn't replicate the full experience of using AT.
